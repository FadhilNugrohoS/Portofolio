Sales Forecasting System for Snack Food Distributors

Sales forecasting system for a snack food distributor is an approach or method to predict the quantity of products that will be sold in the future. The main objective of this system is to assist distributors in managing inventory more efficiently, minimizing the risk of excess or insufficient stock, and overall improving supply chain performance.

The sales forecasting system helps distributors determine the optimal quantity of goods to be stored in the warehouse. By estimating future demand, distributors can avoid excess inventory that may lead to losses due to price reductions or expiration of goods. Conversely, it can also prevent stock shortages that may result in customer loss and missed sales opportunities.

My role in this project is as follows:
1. Create a Database using MySQL.
2. Develop a Sales Forecasting System using the Laravel Framework.
3. Design a Dashboard using the Bootstrap Framework.
4. Implement a Login system using both Laravel and Bootstrap Frameworks.
5. Automatically apply Single Exponential Smoothing Method for Sales Forecasting using the Laravel Framework.
6. Automatically generate Sales Recap Calculations for Each Product per Month from approved orders by the admin using the Laravel Framework.
