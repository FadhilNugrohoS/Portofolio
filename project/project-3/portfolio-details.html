Sistem ERP (Enterprise Resource Planning) adalah perangkat lunak yang dirancang untuk membantu perusahaan mengelola berbagai proses bisnis mereka secara terintegrasi. ERP dapat digunakan dalam berbagai industri, termasuk industri produksi pakaian seperti kaos polos. Dalam konteks produksi kaos polos, sistem ERP dapat menyediakan sejumlah fitur yang berguna untuk mengoptimalkan proses bisnis.

Proyek ini bertujuan untuk mengembangkan sistem ERP untuk perusahaan produksi kaos polos. Sistem ini dapat membantu perusahaan dalam mengelola proses bisnisnya, mulai dari pengelolaan stok bahan baku, pengelolaan pesanan, hingga pengelolaan produksi. Sistem ini dibuat menggunakan framework Laravel dan MySQL sebagai database.

Peran saya dalam proyek ini adalah sebagai berikut:
1.Membuat Database menggunakan MySQL.
2.ERP Produksi Kaos Polos menggunakan framework Laravel.
3.Mendesain Dashboard menggunakan Framework Bootstrap.
4.Membuat sistem pemesanan bahan baku kepada vendor yang memiliki status aktif.
5.Filtering data untuk pemesanan bahan baku terhadap vendor yang memiliki status aktif.
6.Membuat sistem CRUD untuk data BOM (Bill of Material).
7.Membuat sistem CRUD untuk data RFQ (Request for Quotation).
