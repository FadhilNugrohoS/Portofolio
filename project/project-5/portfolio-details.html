Website Apotek Online adalah sebuah website yang dibuat menggunakan framework Laravel. Website ini memiliki fitur CRUD untuk data obat, data pembeli, dan data transaksi. Website ini juga memiliki fitur login dan register untuk admin, guru dan pembeli. Website ini juga menggunakan Spatie Permission untuk manajemen hak akses. pembeli dapat berlangganan dan memilih course yang diinginkan. Pembeli juga dapat melihat course yang telah dibeli dan melihat progress dari course yang diambil.

Peran saya dalam proyek ini adalah sebagai berikut:
1. Membuat desain website
2. Membuat fitur CRUD untuk data obat, data pembeli, dan data transaksi
3. Membuat fitur login dan register untuk admin dan pembeli
4. Membuat fitur manajemen hak akses menggunakan Spatie Permission
5. Membuat fitur berlangganan course
6. Membuat fitur melihat course yang telah dibeli dan melihat progress dari course yang diambil
7. Membuat fitur assessment oleh admin
