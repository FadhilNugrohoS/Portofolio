Sistem tanda tangan elektronik (e-signature) di Kementerian Komunikasi dan Teknologi Informasi (Dinas Kominfo) bertujuan untuk menyederhanakan proses penandatanganan dokumen oleh kepala departemen. Tanda tangan elektronik adalah metode otentikasi digital yang digunakan untuk memvalidasi dokumen atau transaksi secara elektronik. Dalam sistem ini, kepala departemen dapat menandatangani, memberikan catatan revisi, dan menolak penandatanganan dokumen yang diunggah oleh admin.

Peran saya dalam proyek ini adalah sebagai berikut:
1. Membuat Database menggunakan MySQL
2. Mengembangkan operasi CRUD pada Halaman Admin menggunakan framework Laravel
3. Mengimplementasikan operasi CRUD pada Halaman Super Admin menggunakan framework Laravel
4. Mengimplementasikan operasi CRUD pada Halaman Kepala Departemen menggunakan framework Laravel
5. Membuat Halaman Login menggunakan framework Laravel dan Bootstrap
6. Mengembangkan Halaman Unggah PDF menggunakan framework Laravel dan Bootstrap
