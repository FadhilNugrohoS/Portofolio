Website-Based Electronic Signature System

The electronic signature system (e-signature) at the Ministry of Communication and Information Technology (Dinas Kominfo) aims to simplify the document signing process by the head of the department. Electronic signatures are a digital authentication method used to validate documents or transactions electronically. In this system, the department head can sign, provide revision notes, and reject the signing of documents uploaded by the admin

My role in this project is as follows:
1. Create a Database using MySQL
2. Develop CRUD operations on the Admin Page using the Laravel framework
3. Implement CRUD operations on the Super Admin Page using the Laravel framework
4. Implement CRUD operations on the Department Head Page using the Laravel framework
5. Create a Login Page using the Laravel framework and Bootstrap
6. Develop a PDF Upload Page using the Laravel framework and Bootstrap
